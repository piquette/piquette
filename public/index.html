Use Cases

A lot of widely-used or mainstream financial engineering tools are implemented in the classic languages of Java or C++. This leaves a lot of newcomers to the industry with limited options. Implementations of financial data libraries and financial models in modern languages will provide communities with more flexibility. Catalyzing new applications in new languages demand a solid foundation in some of the basic toolsets!
